# RTU Programming Languages – Lab2

## Tasks
- **lab2_1.c** → Sum of numbers 1..n  
- **lab2_2.c** → Factorial (iterative, no recursion)  
- **lab2_3.c** → Prime numbers up to n  

## Build
```bash
make
```
Run
Linux / Mac
```
./lab2/lab2_1
./lab2/lab2_2
./lab2/lab2_3
```
Windows (PowerShell)
```
.\lab2\lab2_1.exe
.\lab2\lab2_2.exe
.\lab2\lab2_3.exe
